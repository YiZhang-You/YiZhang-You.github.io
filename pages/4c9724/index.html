包的细节注意点

# 0. 包的细节注意点

# 1. package进行包的声明，建议：包的声明这个包和所在的文件夹同名

# 2. main函数一定要放在main包下，否则不能编译执行

# 3. 引入包的语法：import "包的路径” 包名是从$GOPATH/src/后开始计算的，使用/进行路径分隔。

# 4. 函数名，变量名首字母大写，函数，变量可以被其它包访问

# 5. 一个目录下不能有重复的函数（可以定义在不同包下）

# 6. 包名和文件夹的名字，可以不一样（但是这个文件夹中的所有文件的包名必须一致）

# 7. 一个目录下的同级文件归属一个包，同级别的源文件的包的声明必须一致

# 8. 可以给包取别名，取别名后，原来的包名就不能使用了
