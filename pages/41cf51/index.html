基本数据类型-布尔类型

# 布尔类型

bool 类型来表示真（对）或假（错），比如常见的5 > 3比较算式，这个是正确的，在程序世界里称之为真（对），Python 使用 True 来代表；再比如4 > 20比较算式，这个是错误的，在程序世界里称之为假（错），Python 使用 False 来代表。

布尔类型可以当做整数来对待，即 True 相当于整数值 1，False 相当于整数值 0

print(True)
print(False)
print(0,type(0))
print(1,type(1))

if 1:
    print("sss")
if not 0:
    print("xxx")

"""
True
False
0 <class 'int'>
1 <class 'int'>
sss
xxx
"""
